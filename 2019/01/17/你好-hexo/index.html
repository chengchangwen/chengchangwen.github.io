流浪地球电影——特效真的很棒，剧情真的很水

发表于 2019-01-17
字数统计: 977 字 | 阅读时长 ≈ 3 分钟

首先原作我没有看，不过据说改动十分的大。

先说说我的观点吧，算得上是中国的第一部制作精良的电影。创意，大幅度的改编，特效，都是从前中国从未有过的，不过，我不认为这是一部优秀的作品，大概7/10吧。

因为我的期待值比较高，我自然会用很高的标准去看待这部电影，我的感觉是，及格水平。

诚然，视觉特效是够了，但是剧情中太多的硬伤实在是有失体验。流浪地球，或者说科幻片，我认为它的精髓在于人，人做了什么事情，事情对人造成了什么影响，或者说对这个电影构造出来的这个“世界”有什么影响。其实就是，你是电影，你得把故事给我讲好了。

我觉得带着地球跑这个创意十分的棒，从来没人这么写过，恋家情结，这是只有中国人能拍出来的电影，我丝毫不反对这一点。

人物设定有问题，先说刘培强父子感情线的问题，这条感情线的描述太少了，丝毫交代不清楚为什么这个年龄段的刘启那么恨自己的父亲，在结尾又表现出那么不想失去父亲，感觉令人迷茫。

然后是刘启的妹妹这个人物设定十分让人出戏，叛逆少女，从头到尾没有发挥任何关键性作用，甚至最后的演讲，明明时间紧迫却还是话那么多，我认为当时任何一个人都可以轻易替代她去做演讲，而且效果要好的多，她的存在在这部电影中就是一个不断需要被救的角色，我认为这个角色存在毫无必要。当然还有中澳合资。

再说回刘启本人，他为什么要往外跑，为什么毫无准备还要带着别人跑，感觉像是为了强行推动剧情发展给出的一个生硬的安排，还有后面莫名其妙的成长，衔接还是觉得生硬，不连贯，让人想不通为什么。

然后就是因为运送火石死掉的军人，我丝毫感受不到他们的死亡所带来的价值。

说这么多其实就是想表达一句话，流浪地球的剧情发展真的铺垫太少太少。

电影的节奏也是有些不太合理，前半段太慢，后半段太快，刻意煽情太多，铺垫太少。

最令人不能接受的就是最后的解决方案就是一个抖机灵想出来的点子加上一句话时间的演算，这个办法真的真的太没有说服力了，还有就是神话程序员的描写，好像什么都能破解。

关于什么物理常识错误之类的东西，我觉得是可以避免，但是不避免也问题不大，背景设定可以相当玄幻，毕竟我看的是人物在这个背景设定下发生的故事。

但是剧情显得还是太做作了。
演技真的尴尬的飞起。

从豆瓣的影评中摘了一句话
:

我希望中国的电影观众会因为浩瀚宇宙的无穷感慨流泪，而不是刻意煽情。

希望各位也能理智的观看这部电影，虽然他是中国科幻电影的里程碑，我个人也比较喜欢这部电影，但是这都不是我们可以过度包容它的理由，中国电影需要进步和批判的眼光，也请各位在赞誉的同时，能够理性的指出片子的不足，中国科幻还有很长的路要走呢。
